Step-by-Step

  1. Go to the Sugar home screen
  2. Open the Write activity
  3. Write a sentence or two
  4. Select some text, and create a heading by increasing the font size and making it bold via the formatting tools
  5. Press the insert image button
  6. Insert one of the pictures you took earlier in Record
  7. Press the main menu button
  8. Save your creation to the Journal as an RTF (Rich Text Format)
