Step-by-Step

  1. Return to the Sugar home screen
  2. Open up the Speak Activity
  3. Make sure the ABC symbol is selected at the top
  4. Type into the white text box and press Enter, the robot will say it aloud!
  5. Now click on the Light Bulb option at the top.
  6. Type in a question (e.g. How far away is the moon?) and press Enter. If able, the robot will answer it aloud!
  7. Feel free to experiment with the other options and change how the robot looks and sounds.
  8. Return to the One Academy Activity
