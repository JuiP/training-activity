The Network View

You can reach the Network view by pressing on your keyboard. Don't do it yet.

There are three kinds of symbol on the Network view:

is a Wi-Fi network (it has a lock if you need a password);

is another XO that you can connect to (more on this later);

is an ad-hoc network (which is used to connect computers to each other without the Internet).

In this case, we are concerned with available Wi-Fi networks.

Hit Next to read about how to connect.
