The Activity Menu

The Activity menu contains tools and options that are specific to the Activity you're using. In some Activities, you might find options for saving to different file formats. For example, in the Write Activity, a document can be saved as plain text (TXT), rich text format (RTF), or portable document format (PDF).

In this case, we're going to take a look at the Description tool.

The Description tool lets us log some information about what we were doing with the current Activity in our Journal (more about that later.)
