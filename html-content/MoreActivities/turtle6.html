Changing the Line

Follow the step-by-step instructions below, or watch the video on the next page.

  1. Use the frame to return to Turtle Blocks
  2. If your blocks are hidden, press the show blocks icon
  3. Press the pen palette icon
  4. Drag and attach a set colour block between the start and forward blocks
  5. Drag and attach a set pen size block under the set colour block
  6. Press the colours palette icon
  7. Drag the colour green to the right hand connector of the set colour block
  8. Click on the number variable next to set pen size, and change it to 15
  9. Clear your canvas by pressing the eraser
  10. Press the snail and watch your turtle draw a green square with thicker lines!
  11. Return to One Academy via keyboard or frame
