Viewing the Journal and Describing an Activity

There are several ways to get to the Journal, but the first one we'll use is on the keyboard.

Once you're there, you're going to learn to view the details of an activity in the Journal, write descriptive text, bookmark the entry as important, and return to the Home view – before coming back to One Academy to learn more!
